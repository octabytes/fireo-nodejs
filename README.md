## Installation

```shell
npm install fireo
```

## Example Usage

```js
const {Model, Field} = require("fireo")

class User extends Model:
    name = Field.Text();


const u = User.init();
u.name = "Azeem Haider";
await u.save()

// Get user
user = await User.collection.get({key: u.key})
console.log(user.name)
```
